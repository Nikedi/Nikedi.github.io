My i3 config

I got my laptop midway through 2019 for school work. Initially i used the pre installed windows on it and installed propably a dozen different linux OSes. Eventually i settled with arch (after like 7 reinstalls). The first window manager I installed was i3 and since then i have maintained the same configuration file.

execs

exec_always --no-startup-id xrandr --auto --output eDP1 --mode 1920x1080 --left-of HDMI-1-1
exec_always --no-startup-id xrandr --auto --output eDP1 --mode 1920x1080 --left-of HDMI-1-0
exec_always --no-startup-id $HOME/.config/polybar/launch.sh
exec --no-startup-id setxkbmap -layout fi
exec --no-startup-id sh ~/.fehbg
exec --no-startup-id xautolock -time 15 -locker 'betterlockscreen -l dimblur'
